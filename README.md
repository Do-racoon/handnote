# Handnote

Handnote 를 이용하면 내가 정리한 글의 이미지를 텍스트로 변환하여 준다.

---------------------------------

## 기능

- 정리한 글의 이미지 → 텍스트

- 하이라이트된 글씨 → 텍스트

- 에디터

- 저장

**[텍스트로 변환]** - 이미지의 모든 글씨 인식

**[중요부분 변환]** - 이미지의 하이라이트한 글씨만 인식
